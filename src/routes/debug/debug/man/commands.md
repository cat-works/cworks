# Commands

This document lists all available commands in the Cat OS Shell. For details on each command, use the manual command: `man <command>`.

---

## System Commands
- `man ipc`

---

## Unix-like Commands
- `man ls`
- `man cd`
- `man mkdir`
- `man cat`
- `man clear`

---

## Filesystem API
- `man stat`
- `man get`
- `man set`

---

## CodeEditor API
- `man load`
- `man save`

---

## Manual Pages
- `man man`
